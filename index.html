Mis Habilidades

Algunas de las tecnologías que he usado y aprendo activamente

HTML5

HTML es un lenguaje de marcas de hipertexto que mediante etiquetas ordenadas forma la estructura de una página web.

CSS3

CSS (hojas de estilo en cascada) estiliza el codigo HTML, referencia, da efectos y diagrama sus elementos (Grid o Flexbox).

JavaScript

JS es el lenguaje orientado a objetos mas usado para crear webs dinámicas, permite realizar efectos, animaciones, acciones, etc.

ReactJs

React es una biblioteca Javascript diseñada para crear interfaces de usuario con el objetivo de facilitar el desarrollo de aplicaciones en una sola página.

Bootstrap

Es un framework que contiene plantillas de diseño, formularios, botones, cuadros, y otros elementos de diseño basado en HTML y CSS para desarrollo Front-End.

Sass

Sass es un metalenguaje de Hojas de Estilo en Cascada (CSS). Es decir un lenguaje de script que es traducido a CSS. Su implementación oficial es open-source y está escrita en Ruby.

MySql

MySQL es un sistema de administración de bases de datos (DBMS) para bases de datos relacionales. Es una aplicación que permite gestionar archivos llamados de bases de datos.

Responsive Design

Responsive Design es una de las técnicas más utilizadas en la actualidad, permitiéndonos hacer que nuestros sitios web puedan adaptarse a cualquier dispositivo móvil o desktop.

Wordpress

WordPress es un CMS:Content Management System (Sistema de gestión de contenidos) desarrollado en el lenguaje PHP, en una de las principales herramientas para la creación de páginas web comerciales.
